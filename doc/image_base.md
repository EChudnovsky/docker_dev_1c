## Базовый образ

Для сборки используется образ от команды LinuxServer.io⁠ [linuxserver/rdesktop](https://hub.docker.com/r/linuxserver/rdesktop/)
Это контейнер, содержащий полноценную среду рабочего стола для Ubuntu, доступную через RDP.
*[далее из документации к образу](https://hub.docker.com/r/linuxserver/rdesktop/)*

## Docker Mode

В состав образа входит фунционал Docker-mods, который обеспечивает дополнительную функциональность в контейнерах без его пересборки. Подключение модов выполняется во время запуска контейнера.

Подродбнее читай [здесь](docker_mode.md)
